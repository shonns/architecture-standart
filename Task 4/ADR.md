### Название задачи:
Интеграция кол-центров с системой актуальных ставок по депозитам

### Автор:
Барнацкий Евгений

### Дата:
23.08.2025

### Функциональные требования

| № | Действующие лица или системы        | Use Case                                 | Описание                                                                                                   |
| :-: | :- | :- | :- |
| 1 | Сотрудник кол-центра | Получение актуальных ставок по депозитам | Сотрудник кол-центра может просмотреть текущие ставки по депозитам в своей системе для консультации клиентов |
| 2 | АБС | Предоставление актуальных ставок | АБС предоставляет актуальные ставки по депозитам через API|
| 3 | Сервис ставок | Генерация файлов со ставками | АБС генерирует файлы с актуальными ставками для партнерского кол-центра |
| 4 | SFTP-сервер | Передача файлов со ставками | Обеспечивает передачу файлов с актуальными ставками партнерскому кол-центру |
| 5 | Партнерский кол-центр | Получение файлов со ставками | Партнерский кол-центр забирает файлы с актуальными ставками через SFTP |
| 6 | Система партнерского кол-центра | Импорт ставок из файлов | Система партнерского кол-центра импортирует ставки из полученных файлов |

### Нефункциональные требования

| № | Требование |
| :-: | :- |
| 1 | Обновление ставок не реже 1 раза в день |
| 2 | Время отклика API для получения ставок не более 500 мс |
| 3 | Шифрование передаваемых файлов со ставками |
| 4 | Резервирование каналов передачи данных |
| 5 | Логирование всех операций передачи ставок |
| 6 | Аутентификация и авторизация доступа к API ставок |
| 7 | Возможность поддерживать разные форматы для файлов со ставками (CSV, JSON) |
| 8 | Возможность отката к предыдущей версии ставок |
| 9 | Уведомление об ошибках передачи данных сотрудникам отдела поддержки |
